Comando Descripción
git init Inicializa un nuevo repositorio de Git.
git clone Clona un repositorio existente.
git add Añade archivos al área de preparación.
git commit Confirma los cambios en el repositorio.
git status Muestra el estado de los archivos en el área de preparación.
git push Envía los cambios al repositorio remoto.
git pull Obtiene y fusiona cambios del repositorio remoto.
git branch Gestiona las ramas del repositorio.
git merge Fusiona ramas en la rama actual.
git log Muestra el historial de confirmaciones.
